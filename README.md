# DC Tech Hub

A production-grade, standalone single-page application (SPA) reference tool for Data Center Technicians.

## Features
- **Aesthetic Design**: Industrial-futuristic dark theme with scan-line effects and glowing UI elements.
- **Encyclopedic Depth**: Detailed full forms, technical specs, and use cases for Networking, Physical Infrastructure, and Power/Cooling.
- **Analytical Diagrams**: Interactive inline SVG diagrams for OSI Models, Rack Units, Aisle Containment, and more.
- **AI-Generated Imagery**: Realistic visual references for data center environments and components.
- **Content Persistence**: LocalStorage-based data injection for adding custom notes and videos.
- **Zero Dependencies**: Entirely built with Vanilla JS, CSS, and HTML.

## Getting Started
### Local Development
To run a local development server with live reload:
```bash
npm install
npm run dev
```

### Deployment
This project is ready to be deployed to **Vercel**, **Netlify**, or **GitHub Pages**. Simply push to a repository and link your hosting provider.
